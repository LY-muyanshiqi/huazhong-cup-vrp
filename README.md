- **Hybrid-ILS 混合迭代局部搜索算法**：三层融合架构（GA 全局探索 + 多算子局部搜索 + 模拟退火自适应接受准则）

## 算法架构

```
Hybrid-ILS
├── 遗传算法 (GA)          → 全局探索，生成优质初始解
├── 多算子局部搜索          → 2-opt / Relocation / Exchange
└── 模拟退火                → 自适应接受准则，防止早熟
```
